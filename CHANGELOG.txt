Beta release.

See [https://next-app.org](https://next-app.org) for more information.

Backend Docker containers:

* **nextappd**: Backend for NextApp
* **signupd**: Handles user sign-up and new devices for existing users

NextApp user application — supported platforms:

* Linux x64 (Flatpak)
* Windows x64 (EXE installer)
* macOS (DMG installers)

The Android version is distributed on [Google Play](https://play.google.com/store/apps/details?id=eu.lastviking.nextapp).
